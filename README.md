## 🛠️ Tech Stack & Architecture
This repository is structured as a fullstack monorepo:
- **Frontend**: React, Vite, TypeScript, Tailwind CSS, Framer Motion, Radix UI.
- **Backend**: Express.js API backend ready for serverless deployment.
- **Form Integration**: Web3Forms for direct email delivery.
- **Deployment & CI/CD**: Fully configured for **Vercel** with **GitHub Actions** workflows enforcing type safety and build success on every commit.

## ⚙️ Running Locally

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Dhruvu2704/Dhruvi_Portfolio.git
   cd Dhruvi_Portfolio
   ```

2. **Install dependencies** (This project uses `pnpm` workspaces):
   ```bash
   pnpm install
   ```

3. **Start the development server:**
   ```bash
   pnpm --filter @workspace/portfolio run dev
   ```
   The site will be running at `http://localhost:5173`.
